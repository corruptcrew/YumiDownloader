## 🚀 **Quick Start**

### **Option 1: Cloud Deploy (Recommended)** ⭐

Click one of the deploy buttons above, configure your `BOT_TOKEN`, and you're done!

### **Option 2: Self-Host** 🖥️

```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/yumi-downloader.git
cd yumi-downloader

# Create virtual environment
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# or
venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Configure
nano config.py

# Run!
python3 main.py
```

---

## ⚙️ **Configuration**

Edit `config.py` with your settings:

```python
# ───────────────────────────────────────────────────────────────
# 🔑 BOT TOKEN (Required)
# ───────────────────────────────────────────────────────────────
BOT_TOKEN = "YOUR_BOT_TOKEN_FROM_BOTFATHER"

# ───────────────────────────────────────────────────────────────
# 👤 YOUR INFO (Shown in "Bot Owner" button)
# ───────────────────────────────────────────────────────────────
DEPLOYER_NAME = "Your Name"
DEPLOYER_USERNAME = "@YourUsername"

# ───────────────────────────────────────────────────────────────
# 📢 FORCE JOIN CHANNELS (Optional)
# ───────────────────────────────────────────────────────────────
DEPLOYER_CHANNELS = []  # Add: ["@yourchannel1", "@yourchannel2"]

# ───────────────────────────────────────────────────────────────
# 💬 SUPPORT & UPDATES (Optional)
# ───────────────────────────────────────────────────────────────
SUPPORT_CHANNEL = "https://t.me/YourSupport"
UPDATE_CHANNEL = "https://t.me/YourUpdates"
```

## 🎨 **Customize Videos**

Replace videos in the `videos/` folder to make the bot yours:

```
videos/
├── 🎬 startmenuvid.mp4      # Welcome video (/start)
├── ⚡ processingvid.mp4     # Processing animation
├── ✅ successvid.mp4        # Download complete
├── 🔒 channelcheckvid.mp4   # Channel join prompt
└── 👤 botownervid.mp4       # Bot owner info
```

**Video Requirements:**
- 📹 Format: MP4
- 💾 Max Size: 20MB (Telegram limit)
- ⏱️ Duration: 5-30 seconds recommended
- 📐 Orientation: Any (landscape or portrait)
